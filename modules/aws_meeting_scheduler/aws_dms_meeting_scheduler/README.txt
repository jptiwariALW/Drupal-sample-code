Drupal Meeting Schedular Module:
------------------------

Overview:
--------
Meeting schedular is a custom, meeting manage for user. Admin can add new meeting
set start and end date, timezone, provide description for meeting.
Admin can view request's for meeting and allow, Deny request. Edit and Delete request's.
Provide block's for easy use.

Features:
---------

The  Meeting Schedular module:

* Provide block named "Meeting Request" for user meeting request form.
* Provide block named "Meeting Scheduler" to show scheduled meeting's by siteadmin.
* E-Mail feature to user and site admin for new meeting request.

Installation:
------------
1. Download and unpack the Libraries module directory in your modules folder
   (this will usually be "sites/all/modules/").
2. Go to "Administer" -> "Modules" and enable the Meeting Schedular module.
3. Go to "Administer" -> "Blocks"  and call the block on requried page.
