Drupal Trip Management Module:
------------------------

Overview:
--------
Trip Management module is developed for displaying trips on front end section.
Trips are displayed the home page, trip listing page and trip detail page.


Installation:
------------
1. Download and unpack the Libraries module directory in your modules folder
   (this will usually be "sites/all/modules/").
2. Go to "Administer" -> "Modules" and enable the Trip Management Module.
